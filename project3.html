Linux Kernel Build & QEMU ARM Simulation

Simulate a Linux operating system on ARM architecture using QEMU

Project Objective

Build and compile a Linux kernel for ARM architecture, then run it on QEMU without real hardware. The goal is to understand the kernel build process, root filesystem management, and simulate ARM board operation via QEMU.

Tools Used

  • QEMU (ARM Emulator)
  • Buildroot (to create rootfs and cross-toolchain)
  • Linux Kernel source (v6.x)
  • ARM Toolchain: `arm-linux-gnueabi-` or `arm-none-eabi-`
  • Ubuntu 22.04 LTS (development environment)

Implementation Steps

  1. Clone the kernel source:
    git clone https://github.com/torvalds/linux.git
  2. Select architecture:
    ARM â€“ use make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- menuconfig
  3. Build the kernel:
    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- zImage
  4. Create rootfs using Buildroot or Busybox
  5. Launch QEMU:
    qemu-system-arm -M versatilepb -kernel zImage \ -initrd rootfs.cpio.gz -append "console=ttyAMA0" -nographic
