Linux Kernel Build & QEMU ARM Simulation

Mô phỏng hệ điều hành Linux trên kiến trúc ARM sử dụng QEMU

Mục tiêu dự án

Xây dựng và biên dịch Linux kernel cho kiến trúc ARM, sau đó chạy kernel đó trên QEMU mà không cần đến phần cứng thực. Mục tiêu là hiểu rõ quy trình build kernel, quản lý root filesystem và mô phỏng bo mạch ARM sử dụng QEMU.

Các công cụ sử dụng

  • QEMU (Giả lập ARM)
  • Buildroot (tạo rootfs và cross-toolchain)
  • Linux Kernel source (v6.x)
  • ARM Toolchain: `arm-linux-gnueabi-` hoặc `arm-none-eabi-`
  • Ubuntu 22.04 LTS (môi trường phát triển)

Quy trình thực hiện

  1. Clone kernel source:
    git clone https://github.com/torvalds/linux.git
  2. Chọn architecture:
    ARM – dùng make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- menuconfig
  3. Build kernel:
    make ARCH=arm CROSS_COMPILE=arm-linux-gnueabi- zImage
  4. Tạo rootfs bằng Buildroot hoặc Busybox
  5. Khởi chạy QEMU:
    qemu-system-arm -M versatilepb -kernel zImage \ -initrd rootfs.cpio.gz -append "console=ttyAMA0" -nographic
