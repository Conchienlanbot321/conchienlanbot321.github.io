Project Portfolio

A curated showcase of embedded systems, IoT, and VLSI design projects

1. Wearable Pulse & SpO2 Monitor

A wearable device using MAX30100 and Arduino Nano to monitor SpO2 and heart rate, with OLED display and PCB designed in KiCad.

2. IoT Environmental Monitoring System

ESP32-based monitoring system with sensors (DHT11, MQ135, water level), sends data to Raspberry Pi and displays on LCD1602 & web interface.

3. Linux Kernel & QEMU ARM Simulation

This project focuses on building a Linux-based embedded system from scratch for an ARM architecture. It involves compiling the Linux kernel, creating a root filesystem using BusyBox, writing an init process, and booting the entire system in QEMU simulating a Cortex-A9 CPU. It also includes manual creation of SD card images, U-Boot replacement, and serial debugging through QEMU’s UART.

4. Processing-in-Memory (PIM) Architecture

This VLSI design project was built based on the architecture of a Processing-in-Memory (PIM) cell, targeting the 90nm CMOS technology. The PIM cell integrates SRAM with logic elements for in-place computation, significantly reducing data movement and energy consumption. The project includes schematic design, full-custom layout, and Cadance simulations for functional and timing validation. It was recognized with an award at the National Smart City IC Design Contest.

5. In-House Security Lock (PIC16F887)

Keypad + LCD-based security lock using PIC16F887 and CCS C compiler. Relay control and real-time feedback on LCD.

6. Multicycle RISC CPU in Verilog

This CPU is implemented in Verilog HDL following a multicycle architecture, separating instruction fetch, decode, execute, memory access, and write-back stages. It features a control unit, register file, ALU, and memory modules, verified using behavioral testbenches and waveforms in Vivado. The design reflects a classical RISC datapath with FSM-based control logic, offering insights into pipelining alternatives and resource-efficient CPU architectures for SoC development.
