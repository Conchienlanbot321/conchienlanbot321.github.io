Design and Construction of In-House Security Lock using PIC16F887

Smart locking system based on PIC microcontroller, keypad, and relay-controlled door lock

Project Objective

Design and build an in-house smart security lock system using PIC16F887 microcontroller, a 4x4 keypad for password input, a relay to control the electronic lock, and an LCD to display system status.

System Components

  • PIC16F887: handles logic, password comparison, relay control, and LCD output
  • 4x4 Keypad: used for entering password
  • 16x2 LCD: displays system status and messages
  • Relay: locks/unlocks door based on password correctness
  • Programming language: C, using CCS C Compiler (.c format)

Working Mechanism

  1. User enters password using keypad
  2. PIC16F887 checks the password against the one stored in EEPROM
  3. If correct: relay is triggered to unlock → LCD displays "ACCESS GRANTED"
  4. If incorrect: relay is not triggered → LCD displays "ACCESS DENIED"
  5. Supports setting a new password via special key combination

Test Results

  • Password input and verification function works reliably
  • Relay responds accurately without delay
  • LCD updates system status in real-time
  • Keypad pin layout was reversed in simulation → fixed in actual wiring
